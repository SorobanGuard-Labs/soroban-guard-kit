# Contributing

Thanks for helping improve Soroban Guard Kit.

## Local setup

```bash
npm install
npm run dev
```

Before opening a PR, run:

```bash
npm run lint
npm run typecheck
npm run test
npm run build
```

## Scope

Prefer focused PRs that improve one workflow: scoring, checklist rules, issue generation, docs, tests, or UI polish.

## Security

Do not publish exploit-ready details for untriaged vulnerabilities. Open a maintainer discussion first and keep public issues scoped to safe remediation work.
