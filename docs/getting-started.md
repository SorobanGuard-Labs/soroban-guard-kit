# Getting started

Install dependencies and run the local app:

```bash
npm install
npm run dev
```

The toolkit runs in demo mode by default. Demo mode uses curated Soroban readiness signals so maintainers can evaluate the workflow before connecting real repositories.

## First review

1. Open `/toolkit`.
2. Check the audit readiness score.
3. Toggle security rules to model resolved or unresolved findings.
4. Review the generated contributor issue.
5. Copy useful checklist items into repository issues or PR review notes.

## Environment

Create `.env.local` from `.env.example` if you want to override public app settings.
